# Real Analysis (MIT OCW 18.100A)

This is Real Analysis (Proof-Based Calculus) covering the Fundamentals of Mathematical Analysis: Real Analysis, Convergence of Sequences and Series, Continuity, Differentiability, Riemann Integral, Sequences and Series of functions, Uniformity, and Interchange of Limit Operations.
* Gain Experience with Proofs.
* Prove Statements about Real Numbers, Functions and Limits.
